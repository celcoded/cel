Monito Monita

MEAN Stack (MongoDB, Express.js, Angular, Node.js), TypeScript, TailwindCSS

A lightweight web application designed to enable a simple and secure Secret Santa experience without unnecessary complexity.

The application intentionally avoids traditional authentication systems. No user accounts, logins, or personal data are required. Instead, users create a group and share a generated group code along with member IDs to grant access, making the onboarding process fast and frictionless.

To improve performance and reduce backend load, the app uses encrypted data exchange between the backend and frontend, allowing client-side interactions to be handled securely without frequent API requests. This design minimizes server calls while preserving data integrity and privacy.

The project demonstrates a focus on clean architecture, efficient data flow, and user-centric design, showcasing practical decision-making around security, performance, and simplicity in a full-stack JavaScript application.
